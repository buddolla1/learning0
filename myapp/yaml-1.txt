💡 How to Use

Open Copilot Chat in VS Code

Switch to Agent Mode

Select the agent you want (e.g., Java Debugging Assistant)

Copy and paste any of these prompts

Agent will provide actionable, code-aware suggestions

⚡ Tip

You can also combine prompts with context, e.g.:

Here is a stack trace from OrderService.java: 
[Paste stack trace here]
Analyze the exception and suggest fixes, including null checks and logging improvements.

Or for SQL:

Analyze the following SQL query for performance and suggest indexing improvements:
SELECT o.id, o.date, u.name FROM orders o JOIN users u ON o.user_id = u.id;
